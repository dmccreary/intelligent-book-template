    Course Description

    This file should contain a detailed description of the course as it would appear in a highly detailed catalog of courses. Students will use this description to determine if the content fits their learning objectives. Other instructors will also use this to ensure that the prerequisites for this course are also taught in other courses. Instructors and curriculum designers will also use this course description to design other advanced data science classes and avoid duplication of content.

    I high-quality course description should also use the 2001 Bloom Taxonomy to create a hierarchy of concepts that should be remembered, understood, applied, analyzed, evaluated and create new projects.

    Title: Book Title Audience College Freshman Duration: 10 weeks

    Prerequisites

    1. Students should know how to use a computer keyboard and mouse
    2. Students should know how to use the copy and paste functions in a text editor
    3. Students should know how to use a web browser and perform actions such as adding a bookmark to a folder
    4. Students should have a GitHub account
    5. Students should be able to install Visual Studio Code (VSCode) on their computer
    6. Students should have access to a generative AI tool such as OpenAI ChatGPT or Anthropic Claude
